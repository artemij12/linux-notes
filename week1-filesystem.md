работа с пользователями и группами:
два варианта создание пользователей:
1.sudo adduser имя пользователя
sudo usermod -a -G название группы имя пользователя добавление пользователя в конец группы
2. sudo useradd - m имя пользователя (с созданием домашнего каталога) sudo passwd имя пользователя

ПРАВА ДОСТУПА
chmod управление правами доступа
первый всегда идет права доступа для владельца, затем группы и остальных
права доступа, владения (chmod,chown,chgr)
chmod изменение прав доступа к файлу или директории
chmod u(владелец)+/-/=rwx(чтение запись выполнение) g(группа)+/-/= o(другие)+/-/=
chown смена владельца и или
chown -R имя пользователя: имя группы директория рекуриснвая смена владельца всей директории
chown имя пользователя: имя группы файл смена владельца файла\
chgrp смена группы без смены владельца альтернативой является chown :group и далее по тексту
chmod можно использовать так же и в виде цифр
7=4+2+1 полный доступ rwx
6=4+2+0 чтение запись
5=4++0+1 чтение и исполнение
